1\. Clone the repository:

  ```
  git clone https://github.com/shk656461/test.git
  ```

2\. Install the dependencies:
  ```
  cd test
  npm install
  ```

3\. Run the development server:
  ```
  npm run dev
  ```

4\. Open localhost:8080 to see the app.

5\. Production build 
  ```
  npm run build
  ```
